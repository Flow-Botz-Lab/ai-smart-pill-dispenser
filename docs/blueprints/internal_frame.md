\# Internal Frame (Fusion 360 Skeleton)



\## Goal

Rigid alignment for carousel, gate, chute, sensors, Pi, and screen.

Bear shell is removable for servicing.



\## Structure

\- Head plate: camera + LED + speaker

\- Vertical spine: main support

\- Carousel plate: supports selector plate + carousel clearance

\- Stepper mount: NEMA 17 pattern (31mm spacing)

\- Gate mount: servo bracket + shutter track

\- Pi + battery plate: Pi standoffs + battery tray

\- Screen mount: supports display behind belly opening



\## Clearances

Keep 2–3mm clearance between frame and bear shell for assembly and wiring.
